A) Introduction - top

BizCraft is a stylish, modern and clietns need html template built with clean and valid HTML5 & CSS3. It based on Twitter Bootstrap framework 3. This template can used for any kind of business, agency and startup company. There are lots of powerful features such as Hero slider, Carousel and Flex Slider, 4 Home variatinos, 2 stylish tab content, portfolio, pricing, blog, Landing Page, Video Background and more. Check out the live demo and enjoy it now !

Template Features

  • 4 Home variation
  • Multipage and OnePage
  • HTML5 Validated
  • Based on Twitter Bootstrap 3
  • Responsive Template
  • FontAwesoe icons
  • Portfolio Isotope
  • Pricing Tables
  • Landing Page
  • Video Background
  • Post Formats
  • Prettyphoto integration
  • Google Maps integration
  • Contact Form
  • Google Web Fonts Lato and JaapokkiRegular
  • Documentation
  • Easy to customize and user friendly

H) FAQs - top

How to change Google map location?

  1. Please, open custom.js file from js folder
  2. Find line 220 in code of this page.
    address: "Corner Swanston St & Flinders St, Melbourne VIC 3000, Australia",
  3. You should change 'Corner Swanston St & Flinders St, Melbourne VIC 3000, Australia' with your own.

How to change Contact Form Email Address for own?

  1. Please, open contact-form.php
  2. Find line 4 you will see
    $recipient = "yourmail@domain.com";
  3. You should change 'yourmail@domain.com' with your own.
